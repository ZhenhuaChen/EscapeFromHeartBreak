# 🚀 快速开始

## 1️⃣ 打开项目（30秒）

### 第一步：打开微信开发者工具
如果还没安装，请访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 第二步：导入项目
1. 点击"导入项目"
2. 项目路径：选择当前文件夹
3. AppID：使用测试号（或你自己的AppID）
4. 点击"导入"

### 第三步：运行
点击"编译"按钮，就可以看到小程序运行了！

---

## 2️⃣ 体验完整流程（3分钟）

### 欢迎页 ✨
看到紫色渐变背景的欢迎页
- 点击"开始你的恢复之旅"

### 输入信息 📝
- 输入你的昵称（随便填）
- 选择性别
- 点击"下一步"

### 回答问题 ❓
回答4个简单问题：
- 分手多久
- 是否还联系
- 是否刷动态
- 睡眠质量

点击"开始恢复计划"

### 主页面 🏠
现在你就进入主页了！可以看到：
- 你的戒断天数（目前是0天）
- 5个戒断任务
- 进度条
- 底部的"情绪急救"和"记录情绪"按钮

试试点击Day 1的任务打卡✓

### 情绪急救 💊
点击底部的"情绪急救"按钮，体验：
- 深呼吸练习（会有动画引导）
- "又想TA了"按钮（会给你建议）

---

## 3️⃣ 测试任务解锁（可选）

默认戒断天数是0天，只能完成Day 1的任务。

**想测试更多任务？**

打开 `app.js` 文件，找到第73行的 `getDetoxDays()` 函数：

```javascript
// 原来的代码
getDetoxDays() {
  const assessment = wx.getStorageSync('assessment')
  if (!assessment || !assessment.startDate) return 0
  const days = Math.floor((Date.now() - assessment.startDate) / (1000 * 60 * 60 * 24))
  return days
}
```

改成：

```javascript
// 测试用：模拟第15天
getDetoxDays() {
  return 15  // 直接返回15
}
```

保存后重新编译，就可以看到Day 14及之前的任务都解锁了！

---

## 4️⃣ 重新开始

想从头体验一遍？

### 方法1：清除缓存（推荐）
在微信开发者工具中：
1. 点击顶部菜单栏的"工具"
2. 选择"清除缓存"
3. 选择"清除数据缓存"
4. 重新编译

### 方法2：控制台清除
在控制台（Console）中输入：
```javascript
wx.clearStorageSync()
```
然后重新编译
